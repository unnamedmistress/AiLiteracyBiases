ÔøΩÔøΩÔøΩ Chain-of-Thought (CoT)

‚úÖ Best for: Math, logic, complex decisions

ÔøΩÔøΩÔøΩ Few-Shot Prompting

‚öôÔ∏è System Prompts vs User Prompts

System Prompt (GPT-5 Custom Instructions)

Sets AI's behavior for all conversations:

"You are a helpful assistant who:\n- Explains complex topics simply\n- Uses analogies and examples\n- Admits when uncertain\n- Cites sources when possible"

User Prompt (Individual messages)

Your specific request:

"Explain quantum computing to a 10-year-old"

ÔøΩÔøΩÔøΩ Prompt Templates Library

Code Review:
"Review this code for:\n1. Security vulnerabilities\n2. Performance issues\n3. Best practices\n4. Readability\n\nProvide specific suggestions with examples."
Content Summarization:
"Summarize in 3 tiers:\n1. One sentence (TL;DR)\n2. 3-5 bullets (key points)\n3. 1 paragraph (detail)"
Creative Brainstorming:
"Generate 10 ideas for [topic]. For each:\n- Brief description\n- Target audience\n- Unique angle\n- Feasibility (1-5)"

Key Takeaways

1

Structure beats luck

System prompts set the character, user prompts set the move. Pair them so the model stays in-role and on-policy.

2

Think in layers

Chain-of-thought, few-shot, and constraint prompts are stackable. Layer them to control reasoning depth, examples, and formatting.

3

Iterate visibly

Ask the model to critique itself, explain deltas, and show intermediate drafts so you can course-correct before final output.
