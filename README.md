## Features

- **Gamified Learning**: XP system, levels, badges, and achievements
- **Dual Modes**: Interactive game mode + detailed presentation/learn mode
- **Interactive Challenges**: Real-world AI scenarios and hands-on activities
- **Progress Tracking**: Track your learning journey across 6 lessons
- **Capstone Project**: Comprehensive final challenge with certificate
- **32 AI Pathologies**: Complete coverage of AI failure modes
- **F.A.C.T.S. Framework**: Practical AI verification methodology

## Complete Course

### Lesson 2: AI Literacy & Bias Detection
- 32 AI pathologies categorized (Hallucination, Confidence, Value/Alignment, Memory)
- Real disaster stories (legal cases, business failures)
- F.A.C.T.S. framework training
- Interactive scenarios and quiz

### Lesson 3: AI Content Creation & Ethics
- AI tools showcase (DALL-E 3, Midjourney, Runway, Suno, ElevenLabs)
- Ethics quiz and case studies
- Prompt builder with quality scoring
- Image, video, audio, and text generation

### Lesson 4: Advanced Prompting Techniques
- Chain-of-Thought (CoT) prompting
- Few-Shot learning and examples
- Role-based prompts and personas
- System prompts vs user prompts
- Prompt tournament competition
- Templates library

### Lesson 5: AI Workflows (Visual)
- Sequential pipelines and parallel processing
- Conditional routing and feedback loops
- Interactive workflow builder
- Real-world automation examples
- Popular tools (Zapier, Make, n8n, LangChain)

### Lesson 6: Capstone Project
- **Deliverable 1**: AI Audit Report
- **Deliverable 2**: Content Strategy Plan
- **Deliverable 3**: Prompt Engineering Portfolio
- **Deliverable 4**: AI Workflow Design
- Certificate of completion (+500 XP)

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- GitHub Pages for hosting
- Responsive design with glassmorphism effects
- LocalStorage for progress persistence
- SVG for visual diagrams

## Learning Outcomes

By completing this course, you will:
- Identify and mitigate 32+ AI pathologies
- Create ethical AI-generated content
- Master advanced prompting techniques
- Design multi-AI workflow systems
- Conduct comprehensive AI audits
- Apply critical thinking to AI outputs
