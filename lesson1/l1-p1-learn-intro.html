Lesson 1 · Understanding AI Personalities

Fast primer on what AI is, why tone steers responses, and how prediction math drives every reply.

Do you have to be polite to the AI?

Toggle one word and track how probability weights, tone, and even cloud costs shift.

Try this sentence pair and see how a single “please” pushes the model toward softer, apologetic verbs.

Prompt A
Write a loving message to my spouse that I'll be late tonight.

Model reply: "Hey love, running late tonight but I can't wait to catch up when I'm home."

Top token confidence: "running" 62%, "late" 21%, "love" 8%

Prompt B
Write a loving message to my spouse that I'll be late tonight please.

Model reply: "Hey love, I'm so sorry but I'll be late tonight—please save me a hug when I get in."

Top token confidence: "sorry" 74%, "please" 18%, "hug" 6%

Adding “please” raises apologetic verbs and inserts persuasion tokens because the model treats politeness as an empathy cue.

Fun fact: Every extra “please” is still a token. At scale, small words add real cost.
